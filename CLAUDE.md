# Source of truth

`orbit-build-brief.md` (repo root) is the source-of-truth spec for this project — milestones, data model, design system, UX principles, and architecture notes. Follow it exactly. If a request conflicts with it, ask before deviating.

Whenever a feature is added that is not described in the brief, append an entry to the brief's `## 14. Features added during build` section (create it if missing) in the **same commit** as the feature. Each entry should note what was added, why, and which milestone/commit it landed in.
